---
title: ToJson()
description: Extract document content as structured JSON with bounding boxes, layout data, and per-block metadata.
---

## Signature

```csharp
public static string ToJson(
    Document   doc,
    List<int>? pages        = null,
    bool       showProgress = false
);

public static string ToJson(
    string     filePath,
    List<int>? pages        = null,
    bool       showProgress = false
);
```

---

## Parameters

<ParamField path="doc" type="MuPDF.NET.Document" required>
  An already-opened MuPDF.NET `Document` instance. Use this overload when calling multiple methods on the same file to avoid re-parsing. Close the document after all operations are complete.
</ParamField>

<ParamField path="filePath" type="string" required>
  Path to a PDF file. This overload opens and closes the document internally. Use for single operations where you do not need to call multiple methods on the same file.
</ParamField>

<ParamField path="pages" type="List<int>?" default="null">
  The pages to include in the output, specified as zero-based page numbers. Pages are processed in the order supplied. If `null`, all pages are processed in document order.
</ParamField>

<ParamField path="showProgress" type="bool" default="false">
  When `true`, writes a per-page progress indicator to the console as pages are processed.
</ParamField>

<Tip>
  `ToJson()` has a narrower parameter surface than `ToMarkdown()` — it does not accept image extraction or OCR options. For OCR-enhanced layout data returned as a typed .NET object, use [`ParseDocument()`](/api-reference/parsed-document) with `useOcr: true`.
</Tip>

---

## Returns

<ResponseField name="string" type="string">
  A JSON string — an array of page objects, one per processed page. See [JSON layout export](/guides/json-layout) for the full field reference and block structure examples.

  The JSON is compact by default. To write indented JSON to disk, round-trip through `System.Text.Json`:

  ```csharp
  using System.Text.Json;

  string json    = PdfExtractor.ToJson("document.pdf");
  var    parsed  = JsonSerializer.Deserialize<object>(json);
  string pretty  = JsonSerializer.Serialize(parsed, new JsonSerializerOptions { WriteIndented = true });

  File.WriteAllText("output.json", pretty, System.Text.Encoding.UTF8);
  ```
</ResponseField>

---

## Exceptions

| Exception | Condition |
|---|---|
| `FileNotFoundException` | `filePath` does not exist or cannot be opened. |
| `MuPdfException` | The file is not a valid PDF, is password-protected, or is corrupted. |

## Examples

### Minimal

```csharp
using PDF4LLM;

string json = PdfExtractor.ToJson("document.pdf");
```

### Iterate over blocks

```csharp
using Newtonsoft.Json.Linq;
using PDF4LLM;

string json  = PdfExtractor.ToJson("document.pdf");
JArray pages = JArray.Parse(json);

foreach (JObject page in pages)
{
    int pageNum = page["page"]!.Value<int>();

    foreach (JObject block in page["blocks"]!)
    {
        if (block["type"]!.Value<string>() != "text") continue;

        foreach (JObject line in block["textlines"] ?? new JArray())
        {
            foreach (JObject span in line["spans"] ?? new JArray())
            {
                var    bbox  = span["bbox"]!.ToObject<float[]>()!; // bounding box for this span
                string text  = span["text"]?.Value<string>() ?? ""; // text content of the span
                int    flags = span["flags"]?.Value<int>() ?? 0;     // font style flags (bitmask)
            }
        }
    }
}
```

### Extract all bounding boxes

```csharp
using Newtonsoft.Json.Linq;

JArray pages = JArray.Parse(PdfExtractor.ToJson("document.pdf"));

foreach (JObject page in pages)
{
    int pageNum = page["page"]!.Value<int>();

    foreach (JObject block in page["blocks"]!)
    {
        string type = block["type"]!.Value<string>()!;
        var    bbox = block["bbox"]!.ToObject<float[]>()!;

        Console.WriteLine($"p{pageNum} {type,-8} bbox=[{string.Join(", ", bbox)}]");
    }
}
```

### Save to file

```csharp
using System.IO;
using PDF4LLM;

string json = PdfExtractor.ToJson("document.pdf");
File.WriteAllText("output.json", json, System.Text.Encoding.UTF8);
```

### Specific pages only

```csharp
string json = PdfExtractor.ToJson(
    "document.pdf",
    pages: new List<int> { 0, 1, 2 }
);
```

### Use alongside ToMarkdown on the same document

```csharp
using MuPDF.NET;
using PDF4LLM;

Document doc = new Document("report.pdf");

string markdown = PdfExtractor.ToMarkdown(doc, writeImages: true, imagePath: "./images");
string json     = PdfExtractor.ToJson(doc);

doc.Close();
```
