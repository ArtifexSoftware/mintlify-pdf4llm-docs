## Overview
 
When `page_chunks=True` is passed to [to_markdown()](../api/to_markdown) or [to_text()](../api/to_text), the return value is a list of dictionaries — one per page — rather than a single concatenated string. Each dictionary follows the schema described on this page.
 
```python
chunks = pymupdf4llm.to_markdown("document.pdf", page_chunks=True)
 
chunk = chunks[0]
print(chunk.keys())
# dict_keys(['metadata', 'toc_items', 'tables', 'images', 'graphics', 'text', 'words', 'page_boxes'])
```
