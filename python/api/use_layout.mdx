## Signature

```python
pymupdf4llm.use_layout(yes: bool = True) 
```

---

## Overview

`use_layout()` switches on/off the use of the PyMuPDF Layout module.

If `yes=True` (default), the layout module will be used for page analysis for optimal results. If `yes=False`, the layout module will not be used.

---

## Parameters

<ParamField path="yes" type="bool" required> 
  Whether to use the layout module. Default is `True`.
</ParamField>
