## Signature

```python
pymupdf4llm.version -> string
```

---

## Overview

`version` returns the version string for PyMuPDF4LLM. This is useful for debugging, bug reports, and confirming which version of the library is installed.

```python
import pymupdf4llm

version = pymupdf4llm.version
print(version)
```

---

## Returns

<ResponseField name="str" type="str">
  The version string for PyMuPDF4LLM.
</ResponseField>

---

## Example Output

```python
"1.27.2.1"
```
